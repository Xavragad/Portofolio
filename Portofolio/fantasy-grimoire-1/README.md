# Fantasy Grimoire

## Overview
The Fantasy Grimoire is a personal portfolio website created by Angelo Roneil C. Paa, showcasing his work as a Game Artist and Developer. The site features atmospheric worlds, characters, and interactive experiences crafted using tools like Blender and Unity.

## Project Structure
The project consists of the following files:

- **index.html**: The main HTML document that contains the structure of the webpage, including sections for the header, about, projects, skills, and contact information.
- **assets/styles.css**: The CSS file that defines the layout, colors, fonts, and responsive design elements to enhance the visual appearance of the grimoire.
- **assets/script.js**: The JavaScript file that includes code for interactive features on the webpage, handling events, animations, or dynamic content updates.

## Setup Instructions
To view the Fantasy Grimoire, follow these steps:

1. Clone the repository or download the project files.
2. Open the `index.html` file in a web browser.
3. Ensure that all assets are correctly linked and accessible.

## Features
- Responsive design for optimal viewing on various devices.
- Interactive elements powered by JavaScript.
- A visually appealing layout with custom styles.
